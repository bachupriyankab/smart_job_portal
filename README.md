# Project Title
Smart Job Portal
## Description

The Smart Job Portal is a web-based application designed to efficiently connect students and recruiters. It allows recruiters to post job opportunities and enables students to create profiles based on their skills.

The system uses an intelligent skill-based job matching mechanism to recommend relevant job opportunities to students. This reduces irrelevant applications, improves hiring efficiency, and provides transparency in the recruitment process through application status tracking.

## Getting Started

## Key Factors
 -->Recruiters can post job openings with requirements
 --> Students can create profiles and add skills  --> Intelligent skill-based job matching
 --> Easy job application system
 --> Recruiters can accept/reject applications
 --> Application status tracking (Pending / Accepted / Rejected)


### Dependencies

Before running this project, make sure you have:

 --> Node.js installed
 --> MongoDB installed and running
 --> Web browser (Chrome/Edge recommended)
 --> Operating System: Windows 10 / Linux / macOS

### Installing
1.Clone the repository:
 -- git clone https://github.com/bachupriyankab/smart_job_portal.git
2.Navigate to project folder:
 -- cd smart_job_portal
3.Install required dependencies:
 -- npm install
4.Make sure MongoDB is running locally

### Executing program

Follow these steps to run the project:

1.Start the backend server:
node app.js
2.Open your browser and go to:
http://localhost:5000/
3.Use the application:
 --> Register as student or recruiter
 --> Add skills (for students)
 --> Post jobs (for recruiters)
 --> Apply and track job status
